 Fitness Tracker

A lightweight single-page fitness tracker for logging daily workouts and hydration.

## Features
- Add exercise entries with sets, reps, and optional notes.
- View daily summary cards for workout count, total sets, and total reps.
- Delete individual workouts or clear the current day.
- Track hydration in 250 ml or 500 ml increments.
- Configure a daily hydration goal and view progress with a visual progress bar.
- Local persistence with `localStorage` so refreshes keep your data.
- Responsive layout for quick use on mobile or desktop.

## Run locally
```bash
python3 -m http.server 4173
```
Then open `http://localhost:4173`.
